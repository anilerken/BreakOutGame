# BreakOutGame
  
  System Verilog implementation of the classical Breakout game. The game runs with BASYS3 FPGA and VGA Monitor.
  There are some problems about the movement of the paddle, probably needs a button synchronizer (used push buttons of FPGA for moving the paddle).
  For more information about the Breakout game: https://en.wikipedia.org/wiki/Breakout_(video_game)
